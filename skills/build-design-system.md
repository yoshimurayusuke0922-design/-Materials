# build-design-system

## 目的

全スライド共通の色、余白、タイポグラフィ、カード、サイドレール、装飾ルールを整理する。

## 入力

- `work/analysis/`
- `work/wireframes/`

## 出力

- `work/design-system.md`

## 手順

1. 赤白基調、サイドレール、カード、薄い赤の有機形状を共通化する。
2. タイトル、リード、本文、注釈のサイズを定義する。
3. カード、ロードマップ、プレースホルダー枠を再利用可能なクラスにする。
4. スライド固有差分は `slide-xx-name` クラスに閉じ込める。
