# create-slide-illustration-assets

## 目的

スライド内で使う小さなイラストやアイコンを、元スライド画像の切り抜きではなく、独立した再利用可能な画像アセットとして作成する。

## 入力

- スライド上の用途
- 必要な意味内容
- 色・線幅・余白などのデザインルール

## 出力

- `out/sanko_html/rebuilt_slides/assets/*.svg`

## 手順

1. 赤線アイコンとして意味が伝わる最小構成を決める。
2. 透明背景のSVGとして作成する。
3. `index.html` では `img` として参照する。
4. アイコン画像内に本文テキストを含めない。
5. 既存スライド画像の切り抜きを使わない。
