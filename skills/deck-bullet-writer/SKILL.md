# Deck Bullet Writer

## Goal

Turn each submessage into short slide-ready bullets.

## Rules

- Use short Japanese phrases that can fit in a slide.
- Avoid awkward line breaks by writing phrases that can stand alone.
- One bullet should carry one idea.
- Avoid arrogant or internal process wording.
- Do not write claims that require unprovided proof.

## Process

1. For each submessage, list 3-5 bullets.
2. Mark bullets as one of:
   - context
   - proposal
   - visual caption
   - risk/condition
3. Keep a separate "do not include" list for risky wording.
4. Store output in `content/bullets.md`.
